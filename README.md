# Install

Clone This Repository

```
$ git clone https://github.com/anasrar/cendek.git && cd cendek
```

Install Dependencies 

```
$ npm install
```

# Run

Build Vue
```
$ npm run build
```

For Development Server

```
$ npm run dev
```

Or

```
$ npm run start
```
